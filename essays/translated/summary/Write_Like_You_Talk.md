# # Write Like You Talk

**글쓰기 개선을 위한 간단한 비법: 말하는 것처럼 쓰기**

더 많은 사람들이 당신의 글을 읽게 하려면 구어체로 써라. 대부분의 사람들은 글을 쓸 때 친구와 대화할 때와는 완전히 다른 언어를 사용한다. 복잡한 문장 구조와 격식적인 단어들을 선택하는데, 이는 독자에게 더 많은 노력을 요구하고 주의를 분산시킨다.

복잡한 아이디어를 표현하는 데 복잡한 문장이 필요하지 않다. 전문가들은 어려운 주제를 논할 때도 일상 대화와 같은 수준의 문장을 사용한다. 비격식적 언어는 "아이디어의 운동복"과 같아서 사고의 흐름을 방해하지 않는다.

개선 방법은 간단하다. 초안을 쓴 후 각 문장을 읽으며 "친구에게 이렇게 말할까?"라고 자문해보라. 아니라면 실제 말하는 방식으로 바꾸면 된다. 출간 전에는 소리 내어 읽어보며 대화처럼 들리지 않는 부분을 수정하라. 이 간단한 원칙만 지켜도 95%의 작가들보다 앞서갈 수 있다.
