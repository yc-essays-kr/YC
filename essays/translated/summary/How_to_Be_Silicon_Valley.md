# # How to Be Silicon Valley

**실리콘밸리 성공 공식: 부자와 너드의 결합**

폴 그레이엄은 실리콘밸리를 다른 곳에서 재현하려면 올바른 사람들, 즉 '부자'와 '너드' 두 종류의 사람만 있으면 된다고 주장한다. 과거 도시들이 수로 근처에 위치해야 성공했던 것과 달리, 현재는 적절한 인재만 확보하면 어디서든 기술 허브를 만들 수 있다.

스타트업 생태계에서 부자(투자자)와 너드(기술자)는 핵심 요소다. 마이애미는 부자는 많지만 너드가 부족하고, 피츠버그는 너드는 많지만 부자가 부족해 모두 스타트업 허브가 되지 못했다. 

정부 관료들은 스타트업 투자자를 대체할 수 없다. 성공한 투자자들은 기술 사업 경험이 풍부하고 개인적 이해관계가 있어 진심으로 관심을 기울이지만, 관료들은 정반대 성향이다. 실리콘밸리의 핵심은 건물이 아닌 사람이며, 단순히 '기술 단지'를 조성하는 것으론 성공할 수 없다.
