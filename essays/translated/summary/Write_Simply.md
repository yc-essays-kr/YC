# # Write Simply

**"Write Simply" 요약**

저자는 평범한 단어와 간단한 문장으로 글을 쓰려고 노력한다고 밝힌다. 단순한 글쓰기의 장점으로는 ▲독자가 읽기 쉬워 내용에 더 깊이 몰입할 수 있고 ▲끝까지 읽을 가능성이 높아지며 ▲독자에게 더 배려적이고 ▲영어가 모국어가 아닌 독자들도 이해하기 쉽다는 점을 제시한다. 

또한 복잡한 문체는 아이디어 부족을 감추는 수단이 될 수 있지만, 단순한 글쓰기는 작가를 정직하게 만든다고 강조한다. 단순한 글은 시간이 지나도 더 오래 지속되는 가치를 가진다.

하지만 저자가 단순하게 쓰는 가장 큰 이유는 복잡한 문장이나 불필요한 지적인 단어 사용이 멋있어 보이는 게 아니라 서투르게 느껴지기 때문이라고 말한다. 그의 글쓰기 방식은 초고를 빠르게 쓴 후 며칠에 걸쳐 편집하며 불필요한 부분을 제거하는 것이다.
