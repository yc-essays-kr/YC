# # What Made Lisp Different

**Lisp의 혁신적 특징들**

1950년대 말 매카시가 개발한 Lisp은 당시 주류였던 Fortran과는 완전히 다른 혁명적인 프로그래밍 언어였다. 

Lisp이 도입한 9가지 핵심 아이디어는 다음과 같다: ①조건문(if-then-else) ②함수를 일급 객체로 취급 ③재귀 함수 ④포인터 기반 변수 개념 ⑤가비지 컬렉션 ⑥표현식 중심 프로그램 구조 ⑦심볼 타입 ⑧심볼 트리를 사용한 코드 표기법 ⑨읽기-컴파일-실행 시간 구분 없이 언어 전체 기능 활용.

이러한 개념들은 1950년대 하드웨어 제약에 묶여 있던 기존 프로그래밍 관습과는 거리가 멀었다. 그러나 시간이 지나면서 주류 언어들이 점차 Lisp의 아이디어들을 채택해왔다. 현재 1~5번 특징은 널리 보급되었고, 6번도 주류에 등장하고 있다. 하지만 8~9번 특징(매크로 시스템의 기반)은 여전히 Lisp만의 고유한 특징으로 남아있다.
