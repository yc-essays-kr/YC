# # Web 2.0

**Web 2.0 요약**

2005년 폴 그레이엄이 쓴 이 에세이는 "Web 2.0"이라는 용어의 의미와 기원을 분석한다. 저자는 처음엔 이 용어가 무의미하다고 생각했지만, 실제로는 복잡한 배경이 있음을 발견했다.

"Web 2.0"은 2004년 O'Reilly와 컨퍼런스 기획사의 브레인스토밍에서 탄생했다. 처음엔 "웹을 플랫폼으로 활용"한다는 의미였지만, 실제로는 웹이 다시 중요해졌음을 강조하려는 마케팅 용어였다. 컨퍼런스는 민주주의를 표방했지만 실상은 벤처캐피털과 대기업 관계자들만 참석할 수 있는 고가의 행사였다.

저자는 2005년 컨퍼런스가 닷컴버블 시절의 분위기와 유사하다고 관찰했지만, IPO 시장이 사라진 상황에서 제2의 버블은 일어나지 않을 것이라고 전망했다. 결론적으로 Web 2.0이라는 용어 자체는 불필요하지만, 웹 기술의 새로운 발전은 실제로 일어나고 있다고 평가했다.
