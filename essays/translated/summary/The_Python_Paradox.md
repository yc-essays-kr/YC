# # The Python Paradox

**파이썬 패러독스 요약**

저자는 '파이썬 패러독스'라는 개념을 제시한다. 이는 상대적으로 널리 알려지지 않은 프로그래밍 언어(파이썬)를 사용하는 회사가 더 뛰어난 프로그래머를 채용할 수 있다는 역설이다.

파이썬을 배우는 사람들은 취업을 위해서가 아니라 순수한 프로그래밍에 대한 열정과 호기심 때문에 새로운 언어를 학습한다. 이런 동기를 가진 프로그래머들이야말로 기업이 원하는 인재들이다. 구글 같은 선진 기업들이 이미 이 원리를 깨닫고 활용하고 있다.

또한 프로그래머에게는 역설적으로 단순히 취업을 위해 배우는 언어가 아닌, 진정한 관심에서 우러나온 언어를 배우는 것이 더 좋은 일자리로 이어진다는 점을 강조한다. 코드의 미적 아름다움도 중요한데, 프로그래머는 코드를 쓰는 시간보다 읽는 시간이 더 많기 때문이다. 결국 프로그래밍을 진정으로 사랑하는 사람들이 만들고 사용하는 언어들이 더 우수한 결과를 낳는다는 것이 핵심 메시지다.
