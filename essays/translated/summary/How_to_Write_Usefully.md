# # How to Write Usefully

**'유용한 글쓰기' 요약**

저자는 설득력보다 '유용성'을 글쓰기의 목표로 제시한다. 유용한 글의 4가지 조건은 다음과 다: ①정확성 - 단순히 맞는 것이 아니라 모호함 없이 명확해야 함 ②강력함 - 거짓이 되지 않는 선에서 최대한 구체적이고 단정적이어야 함 ③중요성 - 독자에게 의미 있는 내용이어야 함 ④참신성 - 독자가 이미 알고 있지 않은 새로운 통찰을 제공해야 함.

이를 달성하는 핵심 전략은 '선택적 발행'이다. 모든 아이디어가 좋을 수는 없지만, 좋지 않은 것은 발행하지 않음으로써 발행하는 모든 글의 질을 보장할 수 있다. 저자는 '느슨하게 쓰고 엄격하게 다듬기' 방식을 제안한다. 초고는 빠르게 다양한 아이디어를 시도하며 쓰되, 이후 며칠간 세심하게 다시 쓰는 과정을 반복한다. 때로는 여러 문단이나 전체 에세이를 폐기하기도 하며, 문장을 수십 번 읽고 수정하는 과정을 통해 진정 유용한 글을 완성한다.
