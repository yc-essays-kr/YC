# # The Island Test

저자는 '섬 테스트'라는 흥미로운 방법을 제시한다. 메인 주 연안의 작은 섬에서 주말을 보낸다고 가정할 때, 옷과 세면용품 외에 반드시 챙기는 물건이 무엇인지 생각해보면 자신이 중독된 것을 알 수 있다는 것이다.

저자 자신의 필수품은 네 가지다. 책은 지루함을 달래는 도피처 역할을 하며, 귀마개는 완전한 정적 속에서 새로운 사고를 위해 필요하다. 노트와 펜은 아이디어를 기록해 다음 생각을 방해받지 않게 하는 전문 도구이자 심리적 안정제 같은 존재다.

저자는 이러한 의존성이 완전히 좋지만은 않다고 인정한다. 특히 책의 경우 현실 도피용 잼 같은 존재라고 표현하며, 가끔 아무것도 가져가지 않고 자신의 생각만으로 지낼 때 오히려 더 창의적이 될 수 있음을 깨달았다고 한다. 이 테스트는 자신의 진짜 필요와 의존성을 돌아보게 하는 유용한 도구다.
