# # The Hundred-Year Language

**요약:**

폴 그레이엄의 "백년 언어"는 100년 후 사용될 프로그래밍 언어를 예측하며, 현재 선택해야 할 언어에 대한 통찰을 제시한다. 

저자는 프로그래밍 언어가 생물종처럼 진화 트리를 형성한다고 주장한다. Cobol처럼 인기를 끌었지만 후손 언어가 없는 '진화적 막다른 길'이 있는 반면, 주류 가지에서 계속 발전하는 언어들이 있다. Java 역시 현재 성공적이지만 Cobol과 같은 운명을 맞을 것이라고 예측한다.

언어 진화는 생물 진화와 달리 가지들이 수렴할 수 있으며, 언어 설계자들이 의도적으로 다른 언어의 아이디어를 통합하기 때문에 돌연변이가 무작위가 아니라는 특징이 있다.

저자는 언어의 핵심을 기본 연산자(공리 역할)와 나머지 부분으로 나누며, 기본 연산자가 언어의 장기적 생존에 가장 중요한 요소라고 강조한다. 수학의 공리처럼 적을수록 좋다는 원칙을 제시한다.
