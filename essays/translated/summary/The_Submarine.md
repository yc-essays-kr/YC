# # The Submarine

**에세이 요약: "잠수함"**

저자는 언론 뒤에 숨어있는 거대한 PR 산업의 실체를 폭로한다. 스타트업을 운영하며 월 1만 6천 달러를 PR 회사에 지불했던 경험을 바탕으로, 전통 매체 기사의 절반 이상이 PR 회사에서 나온다고 주장한다.

PR의 핵심은 거짓말이 아닌 '선택적 진실'이다. 좋은 PR 회사는 기자들에게 진짜 가치 있는 정보를 제공하되, 고객에게 유리한 진실만을 골라서 전달한다. 이는 기자들이 게으르거나 과로에 시달려 직접 취재하기보다 PR 회사가 가져다주는 기사에 의존하기 때문이다.

업계지는 PR 의존도가 높아 보도자료를 거의 그대로 게재하기도 하지만, 뉴욕타임스나 월스트리트저널 같은 일류 매체는 회의적이다. 최고 기자들의 약점은 게으름이 아닌 허영심으로, 마치 그들이 스스로 발견한 이야기처럼 포장해야 한다고 설명한다.
