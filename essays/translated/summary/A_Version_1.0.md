# # A Version 1.0

**에세이 "A Version 1.0" 요약**

폴 그레이엄이 2004년 10월에 쓴 이 에세이는 좋은 글쓰기가 반복적인 수정 과정임을 보여주는 작품입니다. E.B. 화이트의 "좋은 글쓰기는 다시 쓰기"라는 말을 인용하며, 학교에서는 완성된 결과물만 보여주어 학생들이 창작 과정을 오해한다고 지적합니다.

저자는 자신의 에세이 "The Age of the Essay"의 초기 초안을 공개하여 실제 수정 과정을 보여줍니다. 빨간색은 최종 원고에 남은 부분, 회색은 삭제된 부분으로 표시했습니다. 삭제된 내용들은 주로 잘못된 정보, 자랑처럼 보이는 내용, 비판적 표현, 주제에서 벗어난 내용, 어색한 문체, 불필요한 단어들이었습니다.

특히 시작 부분에서 더 많은 수정이 있었고, 방향성이 불분명할 때 곁길로 빠지는 경향이 많았다고 설명합니다. 최종적으로 초안의 3-4분의 1만이 최종 버전에 남았으며, 이는 평균적인 수준이라고 밝힙니다.
