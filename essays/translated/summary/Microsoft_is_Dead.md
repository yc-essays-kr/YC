# # Microsoft is Dead

**마이크로소프트의 몰락 (2007년 4월)**

저자는 젊은 창업가와 대화하며 마이크로소프트에 대한 두려움을 이해하지 못하는 모습을 보고, 마이크로소프트가 이미 죽었음을 깨달았다고 말한다. 1980년대 말부터 20년간 소프트웨어 업계를 지배했던 마이크로소프트는 2005년경 완전히 힘을 잃었다.

마이크로소프트 몰락의 네 가지 원인을 제시한다: **첫째, 구글의 등장**으로 기술업계 패권이 넘어갔다. Gmail 등을 통해 구글이 검색을 넘어선 역량을 보여주며 2005년 주도권을 잡았다. **둘째, 웹 기반 소프트웨어의 부상**이다. Ajax 기술로 데스크톱 애플리케이션 수준의 웹 앱이 가능해지면서 데스크톱 시대가 끝났다. **셋째, 초고속 인터넷의 보편화**로 서버 의존도가 높아지며 데스크톱 필요성이 줄었다. **넷째, 애플의 부활**이다. OS X를 통해 애플이 기술업계에서 극히 드문 부활을 이뤘다.

현재 마이크로소프트는 여전히 수익을 내지만 더 이상 위험한 존재가 아니라고 결론짓는다.
