# # What Microsoft Is this the Altair Basic of?

**에세이 요약**

성공한 스타트업들을 이해하려면 초기에 왜 그렇게 형편없어 보였는지 분석해보라고 저자는 조언합니다. 

마이크로소프트의 Altair Basic 인터프리터, 에어비앤비의 타인 집 에어매트리스 숙박, 페이스북의 대학생 스토킹 사이트 등은 모두 처음에는 단순히 작은 것이 아니라 완전히 잘못된 아이디어처럼 보였습니다. 창업자들조차 왜 자신들의 아이디어가 유망한지 명확히 설명하지 못했고, 단지 본능적으로 미래에 무언가 빠진 것을 감지했을 뿐이었습니다.

저자는 형편없어 보이는 스타트업 아이디어를 접할 때마다 "이것이 어떤 마이크로소프트의 Altair Basic인가?"라고 자문합니다. 이는 조롱 대신 퍼즐로 접근하여 숨겨진 가능성을 찾아보는 방식입니다. 때로는 하나의 스타트업이 여러 개의 서로 다른 거대 기업으로 성장할 수 있는 경로를 가지고 있기도 합니다. 이런 경우 창업자들의 본능을 따르라고 조언합니다.

핵심 메시지는 겉보기에 한심해 보이는 아이디어도 거대한 성공의 씨앗일 수 있다는 것입니다.
