# # Do Things that Don't Scale

**에세이 요약: "확장 불가능한 일들을 하라"**

Y Combinator의 폴 그레이엄이 2013년에 쓴 이 에세이는 스타트업 초기에 수동적이고 비효율적으로 보이는 일들의 중요성을 강조합니다.

**핵심 메시지**: 성공한 스타트업들은 저절로 성장하지 않으며, 창업자가 직접 나서서 성장을 만들어야 합니다. 마치 구형 자동차의 크랭크 손잡이처럼, 처음 시동을 걸기 위해서는 별도의 수고로운 과정이 필요합니다.

**주요 사례**:
- **Stripe**: 베타 테스트 링크만 보내는 대신, 창업자들이 직접 사용자의 노트북을 가져와서 현장에서 설치해주는 "콜리슨 설치" 방식으로 적극적인 사용자 확보
- **Airbnb**: 뉴욕에서 집집마다 방문하여 직접 사용자를 모집하고 기존 사용자들의 리스팅 개선을 도와주는 영업활동

**핵심 인사이트**: 처음에는 절대적 숫자가 작아 보이지만, 주 10% 성장률을 유지하면 복리 성장의 힘으로 1년 후 14,000명, 2년 후 200만 명의 사용자를 확보할 수 있습니다. 초기 스타트업은 매우 취약하며, 직접적인 사용자 참여가 성공과 실패를 가르는 결정적 요소가 됩니다.
