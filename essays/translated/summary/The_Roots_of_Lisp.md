# # The Roots of Lisp

**리스프의 뿌리 요약**

1960년 존 매카시는 프로그래밍 언어 리스프(Lisp)를 발표했는데, 이는 유클리드가 기하학에 한 것과 같은 혁신적 성과였습니다. 매카시는 몇 가지 간단한 연산자와 함수 표기법만으로 완전한 프로그래밍 언어를 구축할 수 있음을 보여줬습니다.

리스프의 핵심 아이디어는 '리스트'라는 간단한 자료구조를 코드와 데이터 모두에 사용한다는 것입니다. 이는 프로그래밍 역사상 획기적인 발견으로, 현재 언어 발전 방향의 모델이 되고 있습니다.

저자는 지금까지 두 가지 일관된 프로그래밍 모델이 있었다고 봅니다: C 모델과 리스프 모델. 컴퓨터 성능이 향상되면서 새로운 언어들은 리스프 모델 쪽으로 발전하고 있으며, 최근 20년간 많은 언어들이 C 모델에 리스프의 특징들(런타임 타이핑, 가비지 컬렉션 등)을 추가하는 방식으로 발전했습니다.

리스프의 독특하고 핵심적인 특징은 자기 자신으로 작성될 수 있다는 점입니다.
