# # If Lisp is So Great

**"Lisp가 훌륭한 언어라면 왜 인기가 없을까?" 요약**

저자는 프로그래밍 언어에서 인기와 품질 사이에는 상관관계가 없다고 주장한다. 존 그리셤이 제인 오스틴보다 많이 팔리지만 더 나은 작가라고 할 수 없듯이, 인기 있는 프로그래밍 언어가 반드시 우수한 것은 아니라는 것이다.

Lisp는 독특한 문법 때문에 어려워 보여서 사람들이 기피한다. 과거에는 "속도가 느리다"는 핑계를 댔지만, 이제는 성능이 개선되자 "다른 언어가 더 인기 있어서"라는 순환논리적 변명을 한다. 

프로그래밍 언어의 인기는 자기강화적 특성이 있다. 인기 언어일수록 더 많은 라이브러리가 개발되고, 기존 프로그램과의 호환성 때문에 바이러스처럼 확산되며, 관리자들도 개발자 교체가 쉬운 인기 언어를 선호한다. 

하지만 모든 프로그래밍 언어가 동등하지 않기 때문에, 제인 오스틴의 소설처럼 덜 인기 있지만 우수한 언어들도 계속 존재한다고 결론짓는다.
