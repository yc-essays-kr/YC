# # A Plan for Spam

**스팸 차단 계획 요약**

2002년 8월에 작성된 이 에세이는 베이지안 통계 기법을 활용한 스팸 필터링 방법을 제시합니다. 

저자는 스팸의 약점이 바로 '메시지 내용'이라고 지적하며, 내용 기반 필터가 스팸 차단의 핵심이라고 주장합니다. 초기에는 "Dear Friend"나 대문자 제목 등 개별 스팸 특성을 인식하는 규칙 기반 접근법을 사용했으나, 정확도 한계와 오탐(false positive) 문제에 직면했습니다.

오탐은 정상 메일을 스팸으로 잘못 분류하는 것으로, 실제 스팸을 놓치는 것보다 훨씬 심각한 문제입니다. 

이후 베이지안 통계 접근법을 도입한 결과, 개별 단어들의 스팸 확률을 조합하는 간단한 알고리즘만으로도 뛰어난 성능을 달성했습니다. 1000개 스팸 중 5개 미만만 놓치고 오탐은 0개라는 성과를 얻었습니다. 흥미롭게도 통계 분석은 "virtumundo", "teens" 같은 명백한 스팸 용어뿐만 아니라 "ff0000"(빨간색 HTML 코드) 같은 예상치 못한 스팸 지표도 발견했습니다.
